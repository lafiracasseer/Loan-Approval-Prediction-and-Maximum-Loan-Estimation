# Loan Approval & Credit Limit Prediction
This project explores the application of Machine Learning in the financial sector to assist analysts in making data-driven decisions regarding loan approvals and credit limits.

# Project Overview
The project is divided into two primary analytical tasks:

Classification: Predicting whether a client’s loan application will be approved or rejected.

Regression: Estimating the maximum loan amount that can be safely offered to a client.

# File Structure
Notebook 1: Data Understanding & Preprocessing (EDA, Handling Missing Values, Label Encoding, and Scaling).

Notebook 2: Classification Modelling (Logic behind feature selection and hyperparameter tuning).

Notebook 3: Regression Modelling (Focusing on Decision Trees and Ensemble methods).

# Tech Stack
Language: Python

Libraries: Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn

Tools: GoogleColab

# Key Findings
Model Performance: Detailed testing was conducted between Decision Tree Regressors (DT-1 vs DT-2).

Feature Importance: Key factors influencing loan approval included Income, Loan-to-Income (LTI) ratio, and Credit History length.

Best Model: The Fully Grown Decision Tree (DT-1) was utilized for final predictions based on its accuracy in estimating credit offers for specific client profiles.
